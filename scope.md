# Digital Shopping Platform — Scope

## In Scope
- Seller homepage: view all items, edit business name, toggle open/close
- Seller item detail: add/edit item name, description, price, contact details
- Customer businesses list page with open/closed badges
- Customer business items page
- Customer individual item page with order form
- REST API for all seller and customer operations

## Out of Scope
- User authentication / login
- Payment integration
- Image uploads
- Order history / tracking
- Search / filtering
- Pagination
